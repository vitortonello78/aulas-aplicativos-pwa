Este repositório contém o projeto desenvolvido durante as aulas de **Frontend com ReactJS** e **Progressive Web Apps (PWA)** na disciplina **Programação de Aplicativos**.

As aulas fazem parte da **2ª fase (2026/1)** do **Curso Técnico em Programação do SENAI/SC**.

O objetivo do projeto é ensinar, de forma **prática e progressiva**, conceitos de desenvolvimento de aplicações modernas utilizando **ReactJS**, evoluindo até a construção de um **Progressive Web App (PWA)**.

---

# 🚀 Tecnologias utilizadas

Este projeto utiliza as seguintes tecnologias:

- ReactJS
- Vite
- JavaScript / TypeScript
- HTML
- CSS
- Progressive Web Apps (PWA)

# 💻 Como rodar o projeto localmente

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/thiagocordeirooo/aulas-aplicativo-pwa.git
```

## 2️⃣ Acesse a pasta do projeto

```bash
cd aulas-aplicativo-pwa
```

## 3️⃣ Instale as dependências

```bash
npm install
```

ou

```bash
npm i
```

## 4️⃣ Inicie o servidor de desenvolvimento

```bash
npm run dev
```

Após iniciar o servidor, abra o navegador no endereço exibido no terminal (normalmente):

```
http://localhost:5173
```

# 🎯 Objetivo educacional

Este projeto foi criado com **fins didáticos**, acompanhando a evolução dos conteúdos apresentados em aula e permitindo que os alunos desenvolvam uma aplicação completa passo a passo.

A ideia é que, ao final da disciplina, os alunos tenham construído um **aplicativo web moderno e instalável**, utilizando boas práticas do desenvolvimento frontend.
